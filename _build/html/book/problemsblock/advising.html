–î–ª—è –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–≥–æ —Ä–µ—à–µ–Ω–∏—è –º–æ–¥–µ–ª–∏ –ò–∑–∏–Ω–≥–∞ –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –º–∞—Ç—Ä–∏—á–Ω—ã–π –ø–æ–¥—Ö–æ–¥. –í —Ä–∞–º–∫–∞—Ö —Ç–∞–∫–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞ —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∞—è —Å—É–º–º–∞ –º–Ω–æ–≥–∏—Ö —Ä–µ—à–µ—Ç–æ—á–Ω—ã—Ö –º–æ–¥–µ–ª–µ–π –ò–∑–∏–Ω–≥–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–∞–ø–∏—Å–∞–Ω–∞ –≤ –≤–∏–¥–µ —Å–ª–µ–¥–∞ –Ω–µ–∫–æ—Ç–æ—Ä–æ–π —Å—Ç–µ–ø–µ–Ω–∏ —Ç—Ä–∞–Ω—Å—Ñ–µ—Ä-–º–∞—Ç—Ä–∏—Ü—ã. –ß—Ç–æ–±—ã –≤—ã—Ä–∞–∑–∏—Ç—å —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫—É—é —Å—É–º–º—É~(2) —á–µ—Ä–µ–∑ —Ç—Ä–∞–Ω—Å—Ñ–µ—Ä-–º–∞—Ç—Ä–∏—Ü—É, —Å–Ω–∞—á–∞–ª–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏–º –≥–∏–±–±—Å–æ–≤—Å–∫—É—é —ç–∫—Å–ø–æ–Ω–µ–Ω—Ç—É –≤ –≤–∏–¥–µ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—è \(N\) –º–Ω–æ–∂–∏—Ç–µ–ª–µ–π

(3)¬∂\[e^{-\frac{E(\{\sigma\})}{\theta}} = T_{\sigma_1 \sigma_2} T_{\sigma_2 \sigma_3} \ldots T_{\sigma_{N-1} \sigma_N} T_{\sigma_N \sigma_1}, \]

–≥–¥–µ

(4)¬∂\[ T_{\sigma \sigma'} = <\sigma|T|\sigma'> = e^{t\sigma\sigma' +\frac{h}{2}(\sigma + \sigma')}, \]

–∞ \(t\) –∏ \(h\) –≤–≤–µ–¥–µ–Ω—ã –¥–ª—è –±–æ–ª–µ–µ –∫–æ–º–ø–∞–∫—Ç–Ω–æ–≥–æ –Ω–∞–ø–∏—Å–∞–Ω–∏—è —Ñ–æ—Ä–º—É–ª –∏ –≤—ã—Ä–∞–∂–∞—é—Ç—Å—è —á–µ—Ä–µ–∑ \(J\), \(H\) –∏ \(\theta\) —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º

\[ t = \frac{J}{\theta},\quad h = \frac{H}{\theta}. \]

–ß–∏—Å–ª–∞ \(T_{\sigma \sigma'}\)~(4) –±—É–¥–µ–º –¥–∞–ª—å—à–µ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å –∫–∞–∫ –º–∞—Ç—Ä–∏—á–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –º–∞—Ç—Ä–∏—Ü—ã \(T\), —Å—Ç—Ä–æ–∫–∏ –∏ —Å—Ç–æ–ª–±—Ü—ã –∫–æ—Ç–æ—Ä–æ–π –≤–º–µ—Å—Ç–æ –Ω–∞—Ç—É—Ä–∞–ª—å–Ω—ã—Ö —á–∏—Å–µ–ª –Ω—É–º–µ—Ä—É—é—Ç—Å—è –∏–∑–∏–Ω–≥–æ–≤—Å–∫–∏–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏ \(\sigma\) –∏ \(\sigma'\). –¢–∞–∫ –∫–∞–∫ –∏–∑–∏–Ω–≥–æ–≤—Å–∫–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –ø—Ä–∏–Ω–∏–º–∞—é—Ç –¥–≤–∞ –∑–Ω–∞—á–µ–Ω–∏—è: \(\sigma, \sigma' = \pm 1\), —Ç–æ –º–∞—Ç—Ä–∏—Ü–∞ \(T\) —è–≤–ª—è–µ—Ç—Å—è –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–π –º–∞—Ç—Ä–∏—Ü–µ–π —Ä–∞–∑–º–µ—Ä–∞ \(2\times 2\). –î–æ–≥–æ–≤–æ—Ä–∏–º—Å—è –¥–∞–ª—å—à–µ —Å—á–∏—Ç–∞—Ç—å, —á—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ \(\sigma = 1\) (\(\sigma' = 1\)) —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –ø–µ—Ä–≤–æ–π —Å—Ç—Ä–æ–∫–µ (–ø–µ—Ä–≤–æ–º—É —Å—Ç–æ–ª–±—Ü—É), –∞ –∑–Ω–∞—á–µ–Ω–∏–µ \(\sigma = -1\) (\(\sigma' = -1\)) —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –≤—Ç–æ—Ä–æ–π —Å—Ç—Ä–æ–∫–µ (–≤—Ç–æ—Ä–æ–º—É —Å—Ç–æ–ª–±—Ü—É). –ú–∞—Ç—Ä–∏—Ü–∞ \(T\) —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏~(4) –≤ —Ç–∞–∫–æ–º —Å–ª—É—á–∞–µ –∏–º–µ–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π —è–≤–Ω—ã–π –≤–∏–¥:

(5)¬∂\[\begin{split} T = \begin{vmatrix} e^{t+h} & e^{-t} \\ e^{-t} & e^{t-h} \end{vmatrix} \end{split}\]

–ú–∞—Ç—Ä–∏—Ü–∞ \(T\) –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ç—Ä–∞–Ω—Å—Ñ–µ—Ä-–º–∞—Ç—Ä–∏—Ü–µ–π –ö—Ä–∞–º–µ—Ä—Å–∞‚Äì–í–∞–Ω–Ω—å–µ.

–†–∞–≤–µ–Ω—Å—Ç–≤–æ~(3) –ø–æ–∑–≤–æ–ª—è–µ—Ç –∑–∞–ø–∏—Å–∞—Ç—å —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫—É—é —Å—É–º–º—É~((2)) –≤ —Å–ª–µ–¥—É—é—â–µ–º –≤–∏–¥–µ

(6)¬∂\[ Z(\theta, H, N) = \sum_{\sigma_1 = \pm 1} \sum_{\sigma_2 = \pm 1} \ldots \sum_{\sigma_{N-1} = \pm 1}\sum_{\sigma_N = \pm 1} T_{\sigma_1 \sigma_2} T_{\sigma_2 \sigma_3} \ldots T_{\sigma_{N-1} \sigma_N} T_{\sigma_N \sigma_1}. \]

–°—Ç—Ä—É–∫—Ç—É—Ä–∞ —Å–ª–∞–≥–∞–µ–º—ã—Ö –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ–ø–∏—Å–∞—Ç—å —Å—É–º–º—É –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

(7)¬∂\[ \sum_{\sigma_1 = \pm 1} \sum_{\sigma_2 = \pm 1} \ldots \sum_{\sigma_{N-1} = \pm 1} \sum_{\sigma_N = \pm 1} T_{\sigma_1 \sigma_2} T_{\sigma_2 \sigma_3} \ldots T_{\sigma_{N-1} \sigma_N} T_{\sigma_N \sigma_1} = \]
\[ \begin{align*} = \sum_{\sigma_1 = \pm 1} \sum_{\sigma_2 = \pm 1} \ldots \sum_{\sigma_{N-1} = \pm 1} T_{\sigma_1 \sigma_2} T_{\sigma_2 \sigma_3} \ldots T_{\sigma_{N-2} \sigma_{N-1}} \sum_{\sigma_N = \pm 1} T_{\sigma_{N-1} \sigma_N} T_{\sigma_N \sigma_1} \end{align*} \]

–¢–µ–ø–µ—Ä—å, –ø–æ—Å–∫–æ–ª—å–∫—É \(T_{\sigma \sigma'}\) —è–≤–ª—è—é—Ç—Å—è –º–∞—Ç—Ä–∏—á–Ω—ã–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –º–∞—Ç—Ä–∏—Ü—ã \(T\), –∑–∞–º–µ—Ç–∏–º, —á—Ç–æ —Å—É–º–º–∞

\[ \sum_{\sigma_N = \pm 1} T_{\sigma_{N-1} \sigma_N} T_{\sigma_N \sigma_1} \]

—Ä–∞–≤–Ω–∞ \(T_{\sigma_{N-1} \sigma_1}^2\), —Ç–æ –µ—Å—Ç—å –º–∞—Ç—Ä–∏—á–Ω–æ–º—É —ç–ª–µ–º–µ–Ω—Ç—É –º–∞—Ç—Ä–∏—Ü—ã \(T^2\), –∫–æ—Ç–æ—Ä–∞—è –ø–æ–ª—É—á–∞–µ—Ç—Å—è –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ (–º–∞—Ç—Ä–∏—á–Ω–æ–≥–æ) —É–º–Ω–æ–∂–µ–Ω–∏—è –¥–≤—É—Ö –º–∞—Ç—Ä–∏—Ü \(T\) (—è–≤–ª—è–µ—Ç—Å—è –∫–≤–∞–¥—Ä–∞—Ç–æ–º –º–∞—Ç—Ä–∏—Ü—ã \(T\)). –¢–æ–≥–¥–∞ —Å—É–º–º–∞ –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞~(7) —Ä–∞–≤–Ω–∞

(8)¬∂\[ \sum_{\sigma_1 = \pm 1}\sum_{\sigma_2 = \pm 1} \ldots \sum_{\sigma_{N-1}} T_{\sigma_1 \sigma_2} T_{\sigma_2 \sigma_3} \ldots T_{\sigma_{N-2} \sigma_{N-1}} T^2_{\sigma_{N-1} \sigma_1}. \]

–î–∞–ª—å—à–µ –∏—Å–ø–æ–ª—å–∑—É–µ–º —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ

\[ \sum_{\sigma_{N-1}} T_{\sigma_{N-2} \sigma_{N-1}} T^2_{\sigma_{N-1} \sigma_1} = T^3_{\sigma_{N-2} \sigma_1} \]

–≥–¥–µ \(T^3 = T\cdot T^2\) ‚Äì –∫—É–± –º–∞—Ç—Ä–∏—Ü—ã \(T\), –∏ –ø–µ—Ä–µ–ø–∏—à–µ–º –≤—ã—Ä–∞–∂–µ–Ω–∏–µ~(8) –≤ –≤–∏–¥–µ

\[ \sum_{\sigma_1 = \pm 1}\sum_{\sigma_2 = \pm 1} \ldots \sum_{\sigma_{N-2}} T_{\sigma_1 \sigma_2} T_{\sigma_2 \sigma_3} \ldots T_{\sigma_{N-3} \sigma_{N-2}} T^3_{\sigma_{N-2} \sigma_1}. \]

–ü—Ä–æ–¥–æ–ª–∂–∏–º —ç—Ç–æ—Ç –ø—Ä–æ—Ü–µ—Å—Å –∏ –ø—Ä–æ—Å—É–º–º–∏—Ä—É–µ–º –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ –ø–æ –≤—Å–µ–º \(\sigma_i\), \(i = 2,\dots,N\), –∫—Ä–æ–º–µ \(\sigma_1\), —Ç–æ–≥–¥–∞ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ~(6) –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç—Å—è –∫ –≤–∏–¥—É

\[ Z(\theta, H, N) = \sum_{\sigma_1 = \pm 1} T^N_{\sigma_1 \sigma_1} \]

–í –ø–æ—Å–ª–µ–¥–Ω–µ–º —Ä–∞–≤–µ–Ω—Å—Ç–≤–µ —Å—É–º–º–∏—Ä—É—é—Ç—Å—è –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –º–∞—Ç—Ä–∏—Ü—ã \(T^N\), –∫–æ—Ç–æ—Ä–∞—è –µ—Å—Ç—å –º–∞—Ç—Ä–∏—Ü–∞ \(T\) –≤ —Å—Ç–µ–ø–µ–Ω–∏ \(N\). –°—É–º–º–∞ –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –º–∞—Ç—Ä–∏—Ü—ã ‚Äì —ç—Ç–æ —Å–ª–µ–¥ –¥–∞–Ω–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –æ–∫–æ–Ω—á–∞—Ç–µ–ª—å–Ω–æ –ø–æ–ª—É—á–∞–µ–º –¥–ª—è —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–æ–π —Å—É–º–º—ã –æ–¥–Ω–æ–º–µ—Ä–Ω–æ–π –º–æ–¥–µ–ª–∏ –ò–∑–∏–Ω–≥–∞ —Å –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ–º –±–ª–∏–∂–∞–π—à–∏—Ö —Å–æ—Å–µ–¥–µ–π –∏ –≤–Ω–µ—à–Ω–∏–º –ø–æ–ª–µ–º, –ø—Ä–∏ –ø–µ—Ä–∏–æ–¥–∏—á–µ—Å–∫–∏—Ö –≥—Ä–∞–Ω–∏—á–Ω—ã—Ö —É—Å–ª–æ–≤–∏—è—Ö, —Å–ª–µ–¥—É—é—â—É—é —Ñ–æ—Ä–º—É–ª—É:

(9)¬∂\[ Z(\theta, H, N) = \mbox{Tr}\, T^N. \]

–û—Ç–º–µ—Ç–∏–º –Ω–µ—Å–∫–æ–ª—å–∫–æ –æ–±—Å—Ç–æ—è—Ç–µ–ª—å—Å—Ç–≤, —Å–≤—è–∑–∞–Ω–Ω—ã—Ö —Å –ø–æ–ª—É—á–µ–Ω–Ω—ã–º –≤—ã—Ä–∞–∂–µ–Ω–∏–µ–º –¥–ª—è —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–æ–π —Å—É–º–º—ã:

  1. –§–æ—Ä–º—É–ª–∞~(9) —è–≤–ª—è–µ—Ç—Å—è —Å–ª–µ–¥—Å—Ç–≤–∏–µ–º —Ç—Ä–∞–Ω—Å–ª—è—Ü–∏–æ–Ω–Ω–æ–π –∏–Ω–≤–∞—Ä–∏–∞–Ω—Ç–Ω–æ—Å—Ç–∏ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞–µ–º–æ–π —Å–∏—Å—Ç–µ–º—ã. –¢—Ä–∞–Ω—Å–ª—è—Ü–∏–æ–Ω–Ω–∞—è –∏–Ω–≤–∞—Ä–∏–∞–Ω—Ç–Ω–æ—Å—Ç—å –≤ –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —ç–Ω–µ—Ä–≥–∏—è —Å–∏—Å—Ç–µ–º—ã —Å–ø–∏–Ω–æ–≤ –Ω–µ –∏–∑–º–µ–Ω—è–µ—Ç—Å—è –ø—Ä–∏ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–æ–º —Å–¥–≤–∏–≥–µ, —Ç–æ –µ—Å—Ç—å —Å–ø–∏–Ω–æ–≤—ã–µ –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏ \(\{\sigma\}\) –∏ \(\{\sigma'\}\), –≥–¥–µ \(\sigma_i' = \sigma_{i+1}\), –∏–º–µ—é—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ —ç–Ω–µ—Ä–≥–∏–∏.

  2. –ü—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –ì–∏–±–±—Å–æ–≤—Å–∫–æ–π —ç–∫—Å–ø–æ–Ω–µ–Ω—Ç—ã –≤ –≤–∏–¥–µ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—è~(3), –≥–¥–µ –º–Ω–æ–∂–∏—Ç–µ–ª–∏ \(T_{\sigma \sigma'}\) –∏–º–µ—é—Ç –≤–∏–¥~(4), –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–º –≤–æ–∑–º–æ–∂–Ω—ã–º. –ú–æ–∂–Ω–æ –≤—ã–±—Ä–∞—Ç—å —Ç—Ä–∞–Ω—Å—Ñ–µ—Ä-–º–∞—Ç—Ä–∏—Ü—É –ø–æ-–¥—Ä—É–≥–æ–º—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –≤–∑—è—Ç—å \(\tilde{T}_{\sigma \sigma'} = e^{t\sigma\sigma' + h\sigma}\), —Ç–æ —Ç–æ–≥–¥–∞: $\( e^{-\frac{E(\{\sigma\})}{\theta}} = \prod^{N}_{j=1} \tilde{T}_{\sigma_{j} \sigma_{j+1}}. \)$ –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –≤ –≤—ã–±–æ—Ä–µ —Ç—Ä–∞–Ω—Å—Ñ–µ—Ä-–º–∞—Ç—Ä–∏—Ü—ã —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –Ω–µ–∫–æ—Ç–æ—Ä—ã–π –ø—Ä–æ–∏–∑–≤–æ–ª. –ó–Ω–∞—á–µ–Ω–∏–µ —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–æ–π —Å—É–º–º—ã –Ω–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —ç—Ç–æ–≥–æ –ø—Ä–æ–∏–∑–≤–æ–ª–∞, –ø–æ—ç—Ç–æ–º—É –∏–º –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω–æ —Ä–∞—Å–ø–æ—Ä—è–¥–∏—Ç—å—Å—è –∏—Å—Ö–æ–¥—è –∏–∑ –∫–∞–∫–∏—Ö-—Ç–æ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö —Å–æ–æ–±—Ä–∞–∂–µ–Ω–∏–π, –Ω–∞–ø—Ä–∏–º–µ—Ä —Ç–∞–∫, —á—Ç–æ–±—ã –º–∞—Ç—Ä–∏—Ü–∞ –æ–±–ª–∞–¥–∞–ª–∞ –∫–∞–∫–æ–π-—Ç–æ —Å–∏–º–º–µ—Ç—Ä–∏–µ–π, —á—Ç–æ —É–ø—Ä–æ—â–∞–µ—Ç –≤—ã—á–∏—Å–ª–µ–Ω–∏—è. –î–ª—è —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–Ω–æ–π –≤—ã—à–µ –æ–¥–Ω–æ–º–µ—Ä–Ω–æ–π –º–æ–¥–µ–ª–∏ –ò–∑–∏–Ω–≥–∞ —Ç—Ä–∞–Ω—Å—Ñ–µ—Ä-–º–∞—Ç—Ä–∏—Ü–∞ –±—ã–ª–∞ –≤—ã–±—Ä–∞–Ω–∞ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω–æ–π —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω–æ–π (—Ç–æ –µ—Å—Ç—å —ç—Ä–º–∏—Ç–æ–≤–æ–π).
