* QMLCourse
Org-repository.

* How-to
** Produce html
#+begin_src shell
pandoc -s --mathjax=https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-chtml-full.js -t html5 input.org -o output.html
#+end_src

** Produce pdf
#+begin_src shell
pandoc --pdf-engine xelatex -f org input.org -o output.pdf
#+end_src
